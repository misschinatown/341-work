Responsive Web Design

Responsive Web Design makes your web page look good on all devices including: desktops, tablets, and phones.

RWD is about using CSS and HTML to resize, hide, shrink, enlarge, or move the content to make it look good on any screen.

Flexbox Layout

Flexbox is a layout model that allows elements to align and distribute space within a container. This layout is an ideal tool when using RWD, creating an optimized experience.
